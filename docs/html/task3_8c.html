Detailed Description

Implements the inorder traversal for printing parse trees.

This module provides the inorder traversal logic used to display the logical formula in its original infix (human-readable) format.

Algorithm: Recursive descent parser

Time Complexity: O(n)

  • Visit each node exactly once
  • Print operation: O(1) per node

Space Complexity: O(n)

  • Recursion call stack: O(h) where h = tree height
  • No additional data structures
