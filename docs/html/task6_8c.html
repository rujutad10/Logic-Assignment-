Detailed Description

Implements logical formula conversion to Conjunctive Normal Form (CNF).

This module eliminates implications, applies De Morgan’s laws, and distributes OR over AND to obtain a valid CNF representation.

Algorithm: Recursive descent parser

Step 1: Eliminate implications (A > B) → (~A + B) Step 2: Move negations inward (De Morgan's laws) Step 3: Distribute OR over AND (convert to CNF)

Time Complexity: O(n)

  • Step 1 (Eliminate >): O(n) single pass
  • Step 2 (Move ~): O(n) single pass
  • Step 3 (Distribute +): O(n × k) tree expansion k = expansion factor, can be exponential
  • WORST CASE: Tree expands from O(n) to O(2^n) nodes

Space Complexity: O(n)

  • Original tree copy: O(n)
  • Expanded tree: O(2^n) in worst case
  • Multiple intermediate trees during transformation
