Detailed Description

Converts fully-parenthesized infix logical expressions into prefix notation.

Uses tokenization, operator precedence, and stack-based processing to safely transform infix expressions into prefix form.

Time Complexity: O(n)

  • Tokenization: O(n)
  • Reversal: O(n)
  • Shunting Yard: O(n)

Space Complexity: O(n)

  • Stack: O(n), Output: O(n), Tokens: O(n)
